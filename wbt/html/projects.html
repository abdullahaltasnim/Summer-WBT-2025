PROJECTS & RESEARCH

JAVA PROJECT

Cafe Management System

Framework: Java (Swing GUI)

Developed a café management system application with a user-friendly GUI using Java Swing. Features include online ordering, digital payment integration, and internal database connectivity for managing menu items, orders, and transactions efficiently.

DATABASE PROJECT

Cinema Hall Management System Database

Database: Oracle Built a cinema hall management database system using Oracle. Designed ER diagrams and implemented DDL and DML queries to manage seat bookings, movie schedules, and payment records. Integrated the Oracle database with an existing Java application via ODBC/JDBC connections to ensure smooth data access and operations.

GRAPHICS PROJECT

A Pixelated View: Cox’s Bazar

Framework: Code::Blocks (OpenGL GLUT), C++

Designed and developed a 3D graphical project in C++ using OpenGL and GLUT within Code::Blocks, showcasing a pixelated model of Cox’s Bazar to demonstrate computer graphics fundamentals and rendering techniques.

C# PROJECT

Framework: C# (.NET)
